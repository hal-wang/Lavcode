# 安全性

对于一款密码管理软件来说，安全性是首要的。使用Lavcode，你管理的密码不用担心被他人获取，你的密码也不会丢失。

# 开源安全

Lavcode 是开源软件，所有代码公开透明。因此你无需担心密码有被窃取的危险。

TIP

源码请访问 GitHubopen in new window 或 Giteeopen in new window

# 备份安全

Lavcode 支持以下备份方式：

  1. GitHub/Gitee 存储
  2. WebDav 云同步
  3. 文件同步

详情看下面备份部分。

# 网络安全

虽然 Lavcode 有联网权限，但只用于以下几处：

  1. 获取 GitHub 相关的源码、评论、通知信息等时
  2. 使用 GitHub/Gitee 存储时
  3. WebDav 同步时（仅与第三方网盘通讯，使用哪个网盘由你自定义）
  4. 从应用商店获取应用信息时，如检查更新、评论等

TIP

如果你懂网络监听技术，会发现Lavcode除以上几条外，完全无任何网络访问行为。
